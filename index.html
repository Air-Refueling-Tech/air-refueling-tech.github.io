Data APIs and web-scraping are used extensively. All the results and plots presented in the pages below aim to be reproducible, referencing the primary-source data directly. The following list contains automatically-updated interactive plots for each category. All content is generated by rendering the Jupyter notebooks into webpages. Automatic updates run once per week.
